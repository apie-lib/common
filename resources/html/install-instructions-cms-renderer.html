Install an existing renderer

Apie does not contain a component renderer by default. The easiest solution is requiring apie/cms-layout-graphite. 

composer require apie/cms-layout-graphite

Creating your own renderer

You neede a class to implement Apie\HtmlBuilders\Interfaces\ComponentRendererInterface (you require to install apie/html-builders with composer. Then in services.yaml you make an alias for Apie\HtmlBuilders\Interfaces\ComponentRendererInterface linking your class. 

services:
    Apie\HtmlBuilders\Interfaces\ComponentRendererInterface:
        alias: MyClassRenderer
